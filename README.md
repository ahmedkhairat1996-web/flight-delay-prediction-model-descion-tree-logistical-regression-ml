Flight Delay Prediction using Machine Learning

 Project Overview
This project develops a machine learning model to predict flight delays using historical airline data. The objective is to support proactive decision-making and improve operational efficiency within the airline industry.

---

 Business Problem
Flight delays create significant challenges for airlines, including:
A-Increased operational costs  
B-Inefficient resource allocation  
C-Reduced customer satisfaction  

This project aims to predict whether a flight will be delayed, enabling better planning and mitigation strategies.

---

 Dataset
A-Source: Kaggle – Flight Delay and Cancellation Dataset (2019–2023)
B-Link: https://www.kaggle.com/datasets/patrickzel/flight-delay-and-cancellation-dataset-2019-2023
C-The dataset includes airline operations data such as flights, airports, and delay indicators across multiple years
D-Due to file size constraints (~600MB), the full dataset is not included in this repository
E-A sample dataset is provided for reproducibility

---

 AI Models DEVOLPED&USED
 A-Logistic Regression (baseline classification model)
 B-Decision Tree (used to capture non-linear relationships and improve prediction performance)

---

 Key Results
A-The models successfully classify flight delays with meaningful predictive accuracy  
B-Decision Tree demonstrated stronger performance in identifying complex delay patterns  
C-Key delay drivers include departure time, airline, and route characteristics  

---

 Business Impact
A-Enables airlines to anticipate delays before they occur  
B-Supports improved scheduling and resource allocation  
C-Reduces operational disruptions and associated costs  
D-Enhances customer experience through proactive communication
